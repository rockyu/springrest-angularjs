TodoApp lets you manage your tasks.

This is a sample application developed to illustrate how to build Single page app(SPA)
using Spring MVC and Angularjs.

Tools used :

  • Spring Mvc as backend rest server
  • Spring roo for initial start
  • Maven as build tool
  • Angularjs as client side framework
  • MySql as database
  • SendGrid for sending emails (feedback module)

Prerequisites :

  • Mysql database
    • Modify database.properties accordingly.
    • Modify hibernate.hbm2ddl.auto field in persistence.xml accordingly.
  • Sendgrid
    • Modify sendgrid.properties for sending emails.

Steps to run :

  • git clone https://github.com/kiranreddykasa/springrest-angularjs.git
  • mvn clean package tomcat:run
  • http://localhost:8080/todoapp/todoes
